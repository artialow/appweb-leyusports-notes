# appweb-leyusports-notes

## 项目简介

本仓库用于归档与发布多个独立 HTML 页面。这些页面通常作为临时说明、工具页面或信息展示页使用，不针对任何特定网站或域名。

仓库名称中的 `leyusports` 为项目内部代号，无实际对外意义。

## 目录说明

```
├── README.md          # 本文件
├── index.html         # 仓库首页（可选）
├── pages/             # 存放所有独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/            # 页面公共资源（CSS、JS、图片等）
```

- `pages/`：主要归档目录，每个 HTML 文件为一个独立页面，互不依赖。
- `assets/`：存放页面间共享的静态资源，如样式文件、脚本或图片。

## 页面归档说明

- 每个 HTML 页面均为自包含的完整文档，可直接在浏览器中打开使用。
- 页面内容以实用信息或功能展示为主，不包含付费推广、外链跳转或敏感信息。
- 页面命名采用 `page-xxx.html` 格式，便于索引与引用。

## 维护说明

- 本仓库由项目成员维护，不定期新增、更新或清理页面。
- 若需添加新页面，请将 HTML 文件放入 `pages/` 目录，并确保其独立可用。
- 请勿在页面中包含任何域名绑定、营销承诺或诱导点击内容。
- 建议为每个页面添加简洁的注释，说明其用途与创建日期。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后在线访问。

## 许可

本仓库内容仅供内部归档与展示使用，不对外提供任何形式的保证或服务。
